# Arare

Arare is a simple license checker on your Ruby project.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'arare'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install arare

## Usage

`arare list ${YOUR_PROJECT_DIR}`
